# Game demo

Проект на Python с использованием Pygame.

## Реализовано

* стартовое меню;
* управление персонажем (WASD);
* система здоровья;
* подсчёт очков;
* таймер;
* враг, преследующий игрока;
* экран победы;
* экран поражения.

## Планируется

* спрайты персонажей;
* несколько уровней;
* анимации;
* музыка и звуковые эффекты;
* разные типы врагов;
* сохранение рекордов.

## Запуск

pip install -r requirements.txt

python main.py
