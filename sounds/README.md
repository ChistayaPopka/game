# Свои звуки

Положи сюда свои аудиофайлы, и игра подхватит их автоматически.
Если файла нет — используется встроенный сгенерированный звук, игра не падает.

## Поддерживаемые форматы
`.ogg`, `.wav`, `.mp3` (порядок поиска именно такой — `.ogg` приоритетнее).

## Куда класть файлы
Игра ищет в двух местах:
1. `assets/` (рядом с картинками)
2. `assets/sounds/` (эта папка) — рекомендуется

## Имена файлов
Для каждого звука подходит любое из имён ниже. Достаточно одного файла.

| Звук в игре   | Имена файлов (любое)                          |
|---------------|-----------------------------------------------|
| Выстрел       | `attack`, `shoot`, `sound_attack`             |
| Подбор монеты | `coin`, `pickup`, `sound_coin`                |
| Подбор оружия | `weapon`, `powerup`, `sound_weapon`           |
| Попадание     | `hit`, `enemy_hit`, `sound_hit`               |
| Урон игроку   | `player_hit`, `damage`, `hurt`, `sound_player_hit` |
| Поражение     | `game_over`, `lose`, `defeat`, `sound_game_over` |
| Победа        | `victory`, `win`, `sound_victory`             |
| Музыка меню   | `menu_music`, `music_menu`                    |
| Музыка игры   | `game_music`, `music_game`                    |

## Пример
```
assets/sounds/coin.ogg
assets/sounds/attack.wav
assets/sounds/game_music.mp3
```
